# Brain-games
### Установка:
   1. make install
### Правила игры: 
   1. Для успешного прохождения игры пользователю необходимо правильно ответить на 3 вопроса подряд.
   2. В случае, если пользователь даст неправильный ответ, игра завершится.
   3. Любой некорректный ввод, принимается за неправильный ответ.
## Игра: "Проверка на четность"
  Пользователю показывается случайное число. Нужно ответить "yes", если число четное и "no", если число нечетное.

  Запускается игра в консоли командой brain-even.

 [![asciicast](https://asciinema.org/a/80wG187DLwLYtG73feYbiLH5p.svg)](https://asciinema.org/a/80wG187DLwLYtG73feYbiLH5p)

## Игра: "Калькулятор" 
 Пользователю показывается случайное выражение, например 13 + 10, которое нужно вычислить.
 
 Запускается игра в консоли командой brain-calc.

 [![asciicast](https://asciinema.org/a/mx8sdT3BYBL9goHJ3gDohTI31.svg)](https://asciinema.org/a/mx8sdT3BYBL9goHJ3gDohTI31)

## Игра: "НОД"
 Пользователю показываются случайные два числа, например 10 и 6. Пользователь должен, вычилить наибольший общий делитель этих чисел.

 Запускается игра в консоли командой brain-gcd.

[![asciicast](https://asciinema.org/a/bGj1JLXoOgdSxlNffwTB73taJ.svg)](https://asciinema.org/a/bGj1JLXoOgdSxlNffwTB73taJ)

## Игра: "Арифметическая прогрессия"
Показывается ряд чисел, образующий арифметическую прогрессию с пропущенным числом. Игрок должен определить и ввести это число.

Игра запускается в консоли командой brain-progression.

[![asciicast](https://asciinema.org/a/Fk0Wu38Di45nXAvH4dB2dEqlr.svg)](https://asciinema.org/a/Fk0Wu38Di45nXAvH4dB2dEqlr)

## Игра: "Простое ли число?"
Пользователю показывают случайное чило. Нужно определить является ли оно простым и ответить "yes", если число является простым и "no", если число не является простым.

Игра запускается в консоли командой brain-prime.

[![asciicast](https://asciinema.org/a/mHag8mwQ89BJ8IN17v7o4neGw.svg)](https://asciinema.org/a/mHag8mwQ89BJ8IN17v7o4neGw)
